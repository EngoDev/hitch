# Hatchway Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a Rust CLI that wraps an arbitrary login command in a PTY, mirrors terminal I/O, detects the first localhost callback URL printed by that command, and starts an SSH reverse tunnel for its port until the wrapped command exits.

**Architecture:** Keep PTY and process orchestration in a small runtime layer and move parsing, detection, resolution, and tunnel command construction into pure modules. Treat tunneling as best-effort support around the wrapped command so tunnel failures never mask the wrapped command's exit status.

**Tech Stack:** Rust 2024, `clap` for CLI parsing, `portable-pty` for PTY management, `url` for URL parsing, `regex` for URL candidate extraction, standard library process and threading primitives.
